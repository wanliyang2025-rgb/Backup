# 研发质量管理全景

## Overview

面向千人规模研发组织的质量管理方法论，覆盖质量评价模型、质量成本、SQA职业困惑与破困路径。

## Key Claims

- **质量评价四维度**：产品质量 / 过程质量 / 持续改进质量 / 成果转化质量 → 9子维度 → 16指标
- **木桶原理**：评价模型识别组织共性问题及能力短板，SQA驱动组织级过程改进
- **质量冰山**：隐藏成本至少是可见成本的 4 倍以上
- **SQA三大困惑**：敏捷时代定位 / 被团队"提防" / 工作成果量化
- **SQA破困三步**：认识自我 → 认准方向 → 认知未来

## 质量红黑榜

| 红（正向） | 黑（负向） | 案例 |
|---|---|---|
| 追根溯源 | 头痛医头 | 代码评审记录糊弄 → 需开会对齐根因 |
| 不忘初心 | 唯指标论 | 50+指标让事业部疲于奔命，质量内卷 |
| 思想先行 | 工具至上 | minitab控制图分析BUG解决时长 → 笑话 |
| 群策群力 | 单打独斗 | 漏测分析只有测试参与 → 不完整 |
| 有始有终 | 虎头蛇尾 | 质量回溯列20条措施 → 真正闭环寥寥无几 |

## 质量成本模型

- **一致成本**：预防成本 + 鉴定成本（→ 提高质量）
- **不一致成本**：内部损失 + 外部损失（→ 质量差）

> 增加预防/鉴定成本 → 质量↑ → 内外部损失↓

**三类主因**：业务软件故障 / 运维遗留系统 / 开发项目失败

```
头脑风暴（发散）→ 亲和图（收敛归纳）→ 矩阵（落地执行：A/B/C/D象限）
```

## SQA破困路径

1. **认识自我**：三环理论 = 擅长 + 热爱 + 机会
2. **认准方向**：管理通道 / 专业通道（流程改进/内控审计/工程能力/管理咨询）
3. **认知未来**：T字型人才 + 培养创造性劳动（AI难替代）
