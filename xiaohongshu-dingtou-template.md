# 小红书定投打卡模板

## 格式规则（2026-03-26 更新）

### 标题格式
- 猪脚饭换算：**每10元收益 = 1碗猪脚饭**
- 赚钱：`昨天赚+XX元（≈X碗猪脚饭）`
- 亏钱：`昨天亏XX元（被教做人）`

### 基金格式（保留明细，精简称）
- 格式：`▫ 简称：金额 → 昨日± / 累计±`
- 简称规则：
  - 华宝标普A股红利机会 → **红利机会**
  - 华夏人工智能ETF联接 → **华夏AI**
  - 天弘标普500(QDII-FOF) → **天弘标普**
  - 大成纳斯达克100ETF联接 → **大成纳指**
  - 南方红利低波50ETF联接 → **红利**
- ⚠️ 简称不超过4字

### 黄金格式
- 格式固定：昨日/累计/持有克数/买入均价→现价/表情

### 100元分配
- 固定5行：黄金ETF/红利/标普500/纳斯达克/AI
- 黄金优先70元，跌多了可加

## 分配原则

- 黄金连续下跌 → 可小幅加仓至70元
- 美股大跌 → 跌多了补，涨了不动
- 单个板块浮亏超10% → 可考虑加码
- 总体原则：越跌越买，越涨越少买
- 100元全投，不留现金
