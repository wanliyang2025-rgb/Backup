# 吴恩达2025 AI趋势总结

## 四大趋势

### 1. 推理模型成为标配

- 2025最大变化："会推理"从少数模型特权 → 成为行业标配
- 关键节点：OpenAI o1（内置推理工作流）、DeepSeek-R1（开源可复现）
- 代价：Gemini 3 Flash 启用推理消耗 1.6亿 Token（未启用仅 740万）
- 核心战场：如何在不牺牲性能的前提下压缩推理成本、提升响应速度

### 2. AI人才争夺战

- Meta点燃战火：小扎亲自游说 + 数亿美元薪酬包
- 人才定价演变：
  - 2011学术期：≈普通工程师
  - 2014-2017商业化：$34.5万/人
  - 2023大模型爆发：$70万+
  - 2025军备竞赛：数亿美元"球星合约"
- 结论：智能本身已成为时代最稀缺的生产资料

### 3. 数据中心：新工业时代

- 标志性事件：2025年AI行业资本支出超3000亿美元
- 超级项目：OpenAI星际之门5000亿、Meta基础投入720亿、亚马逊1250亿
- 麦肯锡预测：到2030年总投资额达5.2万亿美元
- 三大现实忧虑：供需合理性 / 电力供应 / 市场理性回归
- 哈佛经济学家：2025上半年美国GDP增长几乎全部由数据中心和AI投资贡献

### 4. 智能体编程：从辅助到主导

- Devin等编码智能体在同类任务完成率突破80%
- 模型推理为智能体注入"灵魂"：先想清楚再行动，任务分解交给低成本模型执行
- 新行业：Loveable、Replit让零编程经验用户"一键生成"Web应用
- 吴恩达观点：AI取代程序员没必要担忧，善于使用AI的开发者效率提升数量级

## 吴恩达学习建议

> 多上课 + 多动手构建AI系统 + 闲暇读论文

核心三点：
1. **上AI课程**：系统化学习优先于贸然开发（重复造轮子）
2. **动手构建**：很多经验只能通过实践获得，上课≠成为飞行员
3. **读论文**（可选但加分）：最优秀求职者至少偶尔读论文
